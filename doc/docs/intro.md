# Project documentation

This documentation site is built with Docusaurus and published with GitHub Pages.

## What belongs here

- Project overview and architecture notes
- Setup, configuration, and environment variables
- Development and testing workflow
- Deployment and operations guidance
- Troubleshooting notes for common issues

## Local preview

Run the documentation site from the `doc` directory:

```bash
npm install
npm run start
```

Build the static site with:

```bash
npm run build
```
