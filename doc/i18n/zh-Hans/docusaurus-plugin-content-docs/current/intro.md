# 文档

本文档站是理解、运行、测试和维护本仓库的统一入口。

## 目标

建议在这里记录项目用途、安装和配置方法、测试和部署流程、所需环境变量或密钥，以及关键代码、脚本、工作流和运维入口的位置。

## 推荐结构

| 章节 | 建议内容 |
| --- | --- |
| 概览 | 产品目标、目标用户、主要能力和仓库结构。 |
| 启动 | 前置依赖、安装命令、配置说明和本地启动方式。 |
| 开发 | 常用脚本、测试、lint、格式化和贡献流程。 |
| 部署 | 发布流程、GitHub Pages 发布、回滚说明和运行检查。 |
| 运维 | 日志、健康检查、告警、常见故障和恢复步骤。 |

## 本地预览

```bash
cd doc
npm install
npm run start -- --locale zh-Hans
```

构建全部语言版本：

```bash
npm run build
```
