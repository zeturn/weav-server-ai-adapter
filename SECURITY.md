# Security Policy

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report suspected vulnerabilities to the maintainers privately with:

- affected package and version or commit
- steps to reproduce
- expected and actual behavior
- impact assessment, if known
- any suggested mitigation

The maintainers will acknowledge the report, investigate the issue, and coordinate a fix or disclosure plan when needed.

## Supported Versions

This package is in migration bootstrap status. Security fixes should target the active `main` branch until tagged releases are established.
